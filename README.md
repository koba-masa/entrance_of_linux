# entrance_of_linux
Linuxの初心者に対して、触り始めるきっかけになれば。

## 目的
- 基本的なLinuxコマンドの習得
- 基本的なネットワークに関する知識の習得

### 最終目標
- VirtualBoxにて、ウェブサーバとアプリケーションサーバを立ち上げ、ホストマシン外からアクセスできるようにする

![最終目標](/images/readme/goal.png)
